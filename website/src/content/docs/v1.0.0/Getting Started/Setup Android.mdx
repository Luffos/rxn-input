title: Setup Android
---

Once rxn-input was installed, we will manually make a small change to a single file to ensure everything works as expected on Android. This change will be responsible for passing some events from the application's Main Activity to the library so that events can be listen.

## MainActivity.java

```diff
package com.YOUR_APP;

+ import android.util.Log;
+ import android.view.KeyEvent;

+ import com.rxninput.RXNInputModule;

public class MainActivity extends ReactActivity {

+  @Override
+  public boolean onKeyDown(int keyCode, KeyEvent event) {
+    RXNInputModule.onKeyDownEvent(keyCode, event);
+    super.onKeyDown(keyCode, event);
+    return true;
+  }

+  @Override
+  public boolean onKeyUp(int keyCode, KeyEvent event) {
+    RXNInputModule.onKeyUpEvent(keyCode, event);
+    super.onKeyUp(keyCode, event);
+    return true;
+  }

  . . .
```

OBS: This file is this file is usually located at <code>/android/app/src/main/java/.../.../MainActivity.java</code>
