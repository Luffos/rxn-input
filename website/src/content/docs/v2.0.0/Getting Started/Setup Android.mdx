---
title: Setup Android
---

Once rxn-input is installed, you need to manually add some code lines to get things working as expected on Android.

MainActivity.java <br/>( /android/app/src/main/java/.../.../MainActivity.java )


```diff
package com.YOUR_APP;

+ import android.util.Log;
+ import android.view.KeyEvent;

+ import com.rxninput.RXNInputModule;

public class MainActivity extends ReactActivity {

+  @Override
+  public boolean onKeyDown(int keyCode, KeyEvent event) {
+    RXNInputModule.onKeyDownEvent(keyCode, event);
+    super.onKeyDown(keyCode, event);
+    return true;
+  }

+  @Override
+  public boolean onKeyUp(int keyCode, KeyEvent event) {
+    RXNInputModule.onKeyUpEvent(keyCode, event);
+    super.onKeyDown(keyCode, event);
+    return true;
+  }

  . . .
```
